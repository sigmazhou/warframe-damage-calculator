# Warframe Damage Calculator

A web-based damage calculator for Warframe that helps you optimize weapon builds and compare damage output. Calculate DPS with accurate damage formulas, element combinations, critical hits, status effects, and support for riven mods and in-game buffs.

## How to Run

```bash
# Install dependencies
pip install -r requirements.txt

# Start the server
python run_server.py
```

Open `http://localhost:5001` in your browser to use the calculator.
